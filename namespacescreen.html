◆ start_screen()

void screen::start_screen ( vex::brain::lcd & screen,
std::vector< Page * > pages,
int first_page = 0 )

Start the screen background task. Once you start this, no need to draw to the screen manually elsewhere.

start_screen begins a screen. only call this once per program (a good place is vexcodeInit) This is a set and forget type function. You don't have to wait on it or start it in a new thread

Parameters
screenreference to the vex screen
pagesdrawing pages
first_pageoptional, which page to start the program at. by default 0
screenthe brain screen
pagesthe list of pages in your UI slideshow
first_pagethe page to start on (by default 0)
